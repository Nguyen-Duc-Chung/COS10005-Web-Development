Navigation Menu with Active Page Highlight

Each page has a navigation menu with the current page highlighted, enhancing user orientation on the website.

Form validation is implemented on both registration and order pages to ensure correct data entry before submission.

“Same as Delivery Address” Checkbox

This checkbox in the order form allows the billing address to auto-fill based on the delivery address, improving user convenience.

Credit Card Validation Based on Type

Credit card number length is validated based on the card type (Visa, MasterCard, or American Express) on the order page.
